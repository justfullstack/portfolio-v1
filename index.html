ModernMan

  • Django
  • Channels
  • PostgreSQL
  • Javascript
  • AWS S3
  • AWS Lambda
  • Redis
ModernMan is a modern men fashion django e-commerce store complete with a custom authentication system, enhanced with AJAX validation, a cart and three checkout systems for customers to choose from: M-PESA, PayPal and Stripe. The app also features a richly-customized django admin dashboard for store owners, employees and product dispatch service as the store utilizes an external dispatch service. This web app also features a simple chat system to discuss orders implemented using python's asyncio module complementing Django channels. This app was built using a test-driven approach with robust tests for every major functionality.

Athena

  • Javascript
  • Accuweather API
  • Heroku
  • AWS S3

Climate change is a topic that has been on everyone's lips globally. With the weather being ever unpredictable, one could always use a weather tracking and forecast web app to their advantage. That's what Athena was built for. The web app utilizes Accuweather's public API to offer meticulous weather report as well as a forecast for the next five days. The information is available by city.

Mapty

  • Javascript

As a fitness enthusiast, tracking my workouts is at the core of my life. Mapty is a GeoLocation web app that allows stateful tracking of workouts. This allows for a better analysis of a user's workout activity including the location.
